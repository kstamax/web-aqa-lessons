## Файл 1 — `01_variables.py`: Змінні та типи даних

**Концепції:** `str`, `int`, `float`, `bool`, `None`, `type()`, f-рядки.


Файл демонструє, як Python зберігає різні типи даних у змінних і як вставляти змінні в текст через f-рядки.

**Міні-завдання** знаходиться в кінці файлу — виконати самостійно безпосередньо в тому ж файлі.

## Файл 2 — `02_conditions.py`: Умовний оператор

**Концепції:** `if`, `elif`, `else`, оператори порівняння (`==`, `!=`, `>`, `<`, `>=`, `<=`).

Запустити файл:

Файл показує два реальні сценарії: перевірку URL після логіну і перевірку стану кошика. Це не абстрактні приклади — рівно ці перевірки будуть у фінальному скрипті.

**Міні-завдання:** написати умову з чотирма гілками для змінної `score`. Після написання — перевірити кожну гілку, змінюючи значення `score`.

## Файл 3 — `03_loops.py`: Цикли

**Концепції:** `for`, `while`, `range()`, `enumerate()`, `+=`.

Запустити файл:

```bash
python3 03_loops.py
```

`for` перебирає готову колекцію — список товарів. `while` повторює дію, поки виконується умова — додає товари в кошик, зупиняючись на потрібній кількості.

**Міні-завдання:** два незалежних завдання в кінці файлу — одне на `for`, одне на `while`. Виконати обидва.

## Шпаргалка

```python
# Змінні та типи
url = "https://example.com"   # str
count = 3                      # int
price = 9.99                   # float
active = True                  # bool
token = None                   # None

# Виведення
print("Текст")
print(f"Змінна: {url}")

# Умова
if count == 3:
    print("Три")
elif count > 0:
    print("Більше нуля")
else:
    print("Нуль або менше")

# for
for item in ["a", "b", "c"]:
    print(item)

for i, item in enumerate(["a", "b"], 1):
    print(f"{i}. {item}")

# while
n = 0
while n < 3:
    print(n)
    n += 1

# list — впорядкована колекція
products = ["Backpack", "Bike Light", "T-Shirt"]
products.append("Jacket")       # додати в кінець
products.remove("Bike Light")   # видалити за значенням
print(products[0])              # перший елемент
print(products[-1])             # останній елемент
print(len(products))            # кількість елементів

# dict — пари ключ → значення
user = {"username": "standard_user", "role": "admin"}
print(user["username"])         # звернення за ключем
user["is_active"] = True        # додати новий ключ
user["role"] = "viewer"         # змінити значення

# set — унікальні значення, порядок не гарантований
tags = {"smoke", "regression", "smoke"}  # дублікат буде видалений
print(tags)                     # {'regression', 'smoke'}
print("smoke" in tags)          # перевірка наявності: True

# Введення
name = input("Ім'я: ")
print(f"Привіт, {name}!")
```
